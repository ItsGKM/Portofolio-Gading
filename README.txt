# Website Portofolio Interaktif GadingKM

Isi:
- Home
- Tentang Saya
- Keahlian Saya / Bidang Saya
- Kumpulan Penghargaan / Prestasi / Piagam

Cara menjalankan:
1. Ekstrak file ZIP.
2. Buka file `index.html` di browser.
3. Edit tulisan di `index.html` sesuai data asli kamu.
4. Edit warna/tampilan di `style.css`.
5. Edit animasi/interaksi di `script.js`.

Fitur:
- Responsive untuk HP dan laptop
- Dark mode
- Typing text otomatis
- Animasi saat scroll
- Filter kategori penghargaan
- Modal detail penghargaan
