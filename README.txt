CARA MEMAKAI WEBSITE PORTOFOLIO

1. Buka file index.html untuk melihat website.
2. Ubah data pribadi di file data.js.
3. Ganti foto profil:
   - Masukkan foto kamu ke folder assets.
   - Misalnya nama file: foto-saya.jpg
   - Ubah bagian foto di data.js menjadi: assets/foto-saya.jpg
4. Ganti foto piagam:
   - Masukkan foto piagam ke folder assets.
   - Ubah bagian foto pada PRESTASI_KEAHLIAN atau PRESTASI_EKSTRAKURIKULER di data.js.
5. Untuk upload ke GitHub:
   - Upload semua file dan folder ini ke repository GitHub.
   - Aktifkan GitHub Pages dari Settings > Pages.
   - Pilih branch main dan folder /root.

Tema versi ini: Black Luxury dengan aksen emas gelap, abu-abu, dan ungu tua.
